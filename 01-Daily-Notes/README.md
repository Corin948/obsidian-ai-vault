# 01-Daily-Notes

Daily journal entries. Naming convention: `YYYY-MM-DD.md`.

Each daily note should:
- Link to meetings attended that day
- Link to permanent notes created that day
- Track key activities and intentions

---

## Recent Daily Notes

*Use Dataview to list recent entries:*
```dataview
LIST
FROM "01-Daily-Notes"
SORT file.name DESC
LIMIT 7
```
