# 08-Templates

Obsidian templates. Use with the Templater plugin or core Templates plugin.

**Templates:**
- `Inbox-Note.md` — for new captures
- `Daily-Note.md` — for daily journal
- `Meeting-Note.md` — for meeting notes (auto-processed by GitHub Actions)
- `MOC-Template.md` — for new Maps of Content
- `Permanent-Note.md` — for atomic Zettelkasten notes
