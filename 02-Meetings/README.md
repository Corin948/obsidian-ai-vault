# 02-Meetings

Meeting notes. Naming convention: `YYYY-MM-DD-Topic.md`.

When a new raw meeting note is pushed, GitHub Actions will:
1. Extract action items and assign owners
2. Structure the note with proper sections
3. Update the relevant project MOC with new action items

---

## Upcoming & Recent Meetings

```dataview
TABLE date, attendees, type, status
FROM "02-Meetings"
SORT date DESC
```
